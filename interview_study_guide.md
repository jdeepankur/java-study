# Java, Spring Boot & Docker Interview Study Guide

## 1. Overview

This guide is tailored for preparing a backend interview focusing on: -
Java (Core + Java 8+) - Spring Boot - Docker - Microservices & Kafka
basics - AWS & CI/CD basics

------------------------------------------------------------------------

## 2. Java Preparation

### Core Concepts

-   OOP: Encapsulation, Inheritance, Polymorphism, Abstraction
-   Collections: List, Set, Map
-   Exception Handling
-   Multithreading basics

### Java 8+

-   Lambda expressions
-   Streams API (filter, map, collect)
-   Functional interfaces

### Common Questions

-   Difference between HashMap and ConcurrentHashMap
-   What is Optional?

-   What is a Bean?
